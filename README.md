Blessing D3v Portfolio

-A personal portfolio website showcasing my skills and front-end projects. Built with HTML, CSS, and JavaScript, it demonstrates interactive and responsive web applications I’ve developed.

Features

-Showcase of 2 live front-end projects

-Technical skills section with highlighted competencies

-Links to GitHub and LinkedIn profiles

-Contact button for inquiries

-Fully responsive design for desktop and mobile

Technologies Used

-HTML5 – structure and content of the website
-CSS3 – styling, layout, and responsive design
-JavaScript – interactive elements and DOM manipulation

How to Use:

Prerequisites
-Modern web browser (Chrome, Firefox, Edge, or Safari)

Clone the repository-
git clone https://github.com/Blessing-D3velop/Blessing-D3v-Portfolio.git
Open Locally

Open index.html in your browser to view the portfolio.
